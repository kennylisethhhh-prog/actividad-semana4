Sistema operativo:
Es el software principal del computador o servidor. Se encarga de administrar los recursos del sistema y permitir la ejecución de programas como Node.js.

Node.js:
Es un entorno de ejecución que permite ejecutar código JavaScript fuera del navegador. Gracias a él, podemos crear y administrar servidores web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas, las respuestas y la forma en que la aplicación atiende las solicitudes de los usuarios.

Puerto:
Es el número por el cual el servidor escucha y recibe las conexiones. En esta práctica se utiliza el puerto 3000 para acceder a la aplicación desde el navegador.

Logs:
Son registros que almacenan información sobre la actividad y los eventos del servidor. Ayudan a monitorear el funcionamiento y detectar posibles errores.
